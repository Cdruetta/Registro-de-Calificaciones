# Mi Primeros Códigos - Registro de Calificaciones

Este proyecto permite registrar calificaciones en el Local Storage del navegador. Más adelante, se podría conectar a una base de datos para llevar un registro mensual o anual.

## Tecnologías Utilizadas

- **HTML**
- **JavaScript**
- **CSS**

## Descripción

Este proyecto es una aplicación web sencilla que permite a los usuarios registrar y gestionar calificaciones. Actualmente, las calificaciones se almacenan en el Local Storage del navegador, pero en futuras versiones, se planea integrar una base de datos para mantener un registro más robusto y a largo plazo.

## Funcionalidades

- **Registrar Calificaciones:** Permite ingresar y guardar calificaciones en el Local Storage.
- **Visualizar Calificaciones:** Muestra las calificaciones almacenadas.
- **Eliminar Calificaciones:** Permite borrar calificaciones del Local Storage.

## Instalación

Para ejecutar este proyecto en tu máquina local, sigue estos pasos:

1. **Clona este repositorio:**
    ```bash
    git clone https://github.com/tu-usuario/mi-primeros-codigos.git
    ```

2. **Navega al directorio del proyecto:**
    ```bash
    cd mi-primeros-codigos
    ```

3. **Abre `index.html` en tu navegador:**
    Simplemente abre el archivo `index.html` en cualquier navegador web para ejecutar la aplicación.

## Uso

1. **Ingresar Calificaciones:**
   - Usa el formulario en la página principal para agregar nuevas calificaciones.
   - Las calificaciones se guardarán automáticamente en el Local Storage.

2. **Visualizar Calificaciones:**
   - Las calificaciones guardadas se mostrarán en una lista en la página principal.

3. **Eliminar Calificaciones:**
   - Haz clic en el botón "Eliminar" junto a cualquier calificación para eliminarla del Local Storage.
